# Switchboard

Open-source, telephony-first AI voice agents with built-in evals,
cost-aware routing, and multilingual support.

Most open-source voice agent frameworks are WebRTC-first and bolt on
telephony later. Switchboard inverts that: PSTN/SIP from day one,
because that's where real businesses live. Ships with an eval harness,
per-call cost tracking, and first-class support for English/Hindi/Urdu
code-switching.
